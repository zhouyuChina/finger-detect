# 专利点挖掘文档

## 项目概述

本项目是一个基于微信小程序的指甲健康检测应用，通过拍照进行灰指甲等指甲疾病的AI智能检测，并提供检测记录管理、恢复进度追踪等功能。

---

## 一、核心技术专利点

### 1. 智能相机引导与图像采集系统

**专利点名称**：一种基于视觉引导框的指甲图像智能采集方法及系统

**技术要点**：
- **视觉引导框定位技术**：通过预设的引导框（280rpx × 360rpx）在相机预览中精确定位，引导框位于屏幕中央上方46%位置，采用镂空遮罩技术突出目标区域
- **多条件实时检测引导**：
  - 光线检测：确保自然光充足
  - 背景检测：确保背景无杂物
  - 位置检测：确保指甲位于引导框内
- **自适应屏幕适配**：根据设备屏幕尺寸（rpx转px）动态计算引导框位置和大小
- **实时视觉反馈**：通过遮罩层（透明度85%）和引导图标提供实时拍照指导

**创新点**：
1. 将复杂的医学图像采集要求转化为直观的视觉引导系统
2. 多条件检测的实时反馈机制
3. 镂空遮罩技术提升用户体验

**技术实现位置**：
- `pages/camera/camera.wxml` - 引导框UI实现
- `pages/camera/camera.wxss` - 遮罩和引导框样式
- `pages/camera/camera.js` - 相机控制和授权管理

### 2. 智能图像自动裁剪技术

**专利点名称**：一种基于引导框坐标的医学图像自动裁剪方法

**技术要点**：
- **坐标映射算法**：建立相机预览视图、屏幕坐标、实际拍摄图片之间的三维坐标映射关系
- **缩放比例计算**：
  - 根据屏幕宽高比和图片宽高比计算缩放因子
  - 处理图片宽高比与屏幕比例不一致的情况（以高度或宽度为准的缩放策略）
- **偏移量补偿**：计算图片在相机视图中的偏移量，确保裁剪区域准确对应引导框位置
- **边界保护**：裁剪区域超出图片边界时的保护机制（Math.max/Math.min限制）

**算法核心**：
```javascript
// 计算缩放比例
if (imageRatio > screenRatio) {
  // 图片更宽，以高度为准
  scaleY = imageHeight / windowHeight
  scaleX = scaleY
  offsetX = (imageWidth - screenWidth * scaleX) / 2
} else {
  // 图片更高，以宽度为准
  scaleX = imageWidth / screenWidth
  scaleY = scaleX
  offsetY = (imageHeight - windowHeight * scaleY) / 2
}

// 计算裁剪区域
actualCropX = guideFrameX * scaleX + offsetX
actualCropY = guideFrameY * scaleY + offsetY
```

**创新点**：
1. 解决了移动端相机预览与实际拍摄图片的坐标不一致问题
2. 自动裁剪技术减少用户操作步骤
3. 精确的坐标映射算法确保裁剪准确性

**技术实现位置**：
- `pages/camera/camera.js` - `cropPhoto()` 方法

### 3. 多用户多档案管理系统

**专利点名称**：一种支持家庭成员管理的健康档案多层级管理系统

**技术要点**：

  - 右手：大拇指、食指、中指、无名指、小指
  - 左脚：大脚趾、第二脚趾、第三脚趾、第四脚趾、小脚趾
  - 右脚：大脚趾、第二脚趾、第三脚趾、第四脚趾、小脚趾
- **档案与检测记录的关联**：每个档案可关联多条检测记录，支持时间序列追踪
- **权限管理**：区分本人档案和其他用户档案，支持档案共享查看

**创新点**：
1. 将单一用户的健康管理扩展为家庭健康管理
2. 精细化的身体部位分类体系
3. 档案与检测记录的多对多关联关系

**技术实现位置**：
- `pages/create-profile/create-profile.js` - 档案创建和管理

**创新点**：
1. 多级分类体系提高检测准确性
2. 置信度机制增强结果可信度
3. 智能后处理提升用户体验

### 7. 图片预处理与格式转换系统

**专利点名称**：一种医学图像的自动格式识别与转换方法

**技术要点**：
- **格式自动识别**：根据文件扩展名自动识别图片格式（JPEG、PNG等）
- **MIME类型映射**：自动将文件扩展名映射为对应的MIME类型
- **Base64编码转换**：将本地图片文件转换为Base64编码，便于网络传输
- **格式统一化**：统一转换为后端可识别的格式

**技术实现**：
```javascript
// 根据文件扩展名确定MIME类型
const extension = imagePath.split('.').pop().toLowerCase()
let mimeType = 'image/jpeg' // 默认
if (extension === 'png') {
  mimeType = 'image/png'
} else if (extension === 'jpg' || extension === 'jpeg') {
  mimeType = 'image/jpeg'
}

const base64Data = `data:${mimeType};base64,${res.data}`
```

**创新点**：
1. 自动格式识别减少用户操作
2. 统一的格式转换接口
3. 支持多种图片格式

**技术实现位置**：
- `pages/photo-detection/photo-detection.js` - `convertImageToBase64()` 方法
