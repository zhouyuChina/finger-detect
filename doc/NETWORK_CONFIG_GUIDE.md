# 微信小程序网络配置指南

## 概述

微信小程序在真机上访问外部接口需要正确配置域名白名单和网络设置。

## 问题分析

### 常见错误
- `ERR_CONNECTION_REFUSED`：连接被拒绝
- `errno: 600001`：网络错误
- 原因：域名未配置或网络设置问题

## 解决方案

### 1. 微信小程序后台配置

#### 登录微信公众平台
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 登录您的小程序账号

#### 配置服务器域名
1. 进入 **开发** → **开发管理** → **开发设置**
2. 找到 **服务器域名** 配置区域
3. 添加以下域名到 **request合法域名**：

```
http://47.76.126.85:4000
```

#### 配置上传域名（如果需要）
```
http://47.76.126.85:4000
```

### 2. 本地开发配置

#### 开发工具设置
1. 打开微信开发者工具
2. 进入 **详情** → **本地设置**
3. 勾选 **不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书**

### 3. 真机测试配置

#### 真机调试设置
1. 在开发者工具中点击 **预览**
2. 使用微信扫码在真机上打开
3. 确保真机网络正常

#### 网络环境要求
- 确保手机网络连接正常
- 如果使用公司网络，确保防火墙允许访问
- 如果使用家庭网络，确保路由器设置正常

## 配置检查清单

## 故障排除

### 1. 检查服务器状态
```bash
curl -I http://47.76.126.85:4000/api/miniprogram/auth
```

### 2. 检查网络连接
```bash
ping 47.76.126.85
```

### 3. 检查端口是否开放
```bash
telnet 47.76.126.85 4000
```

### 4. 常见错误及解决方案

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| ERR_CONNECTION_REFUSED | 服务器未启动或端口关闭 | 检查服务器状态 |
| errno: 600001 | 网络连接失败 | 检查网络设置和域名配置 |
| 域名不在白名单 | 未配置合法域名 | 在微信公众平台添加域名 |

## 注意事项

1. **开发环境**：可以使用 HTTP 协议
2. **生产环境**：必须使用 HTTPS 协议
3. **域名配置**：需要在小程序后台配置合法域名
4. **网络限制**：微信小程序有严格的网络访问限制
5. **调试模式**：开发工具可以忽略域名限制，但真机不行
